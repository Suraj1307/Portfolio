# Suraj Kumar Portfolio

Static portfolio website based on the visual structure of `jigar-sable/Portfolio-Website`, customized with Suraj Kumar's content, projects, resume, and contact details.

## Run locally

Run with Node.js from the project root:

```bash
npm install
npm start
```

Then open `http://localhost:3000` locally.

## Render web service

If you want the previous Render web-service deployment style, use:

- Build Command: `npm install`
- Start Command: `npm start`
